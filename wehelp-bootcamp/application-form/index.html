Q. 個人簡介。

我喜愛探究事物行動的原因與背後的原理，喜歡分析和釐清資訊。擅長處理細節、分析流程、熟悉工具用法與應用，將流程分解後尋找方法建構。

在準備轉職之前，我從事展場場務、室內裝修等勞動力工作的自由工作業。

以前，我會接受各式各樣的工作，只要是我能夠勝任的。但是，最近一段時間，我開始認真思考自己想要從事什麼樣的工作，並查閱了很多相關資料。

最終，我決定下一步要成為一名軟體工程師。

肯定還有許多困難等待著我，但我期待能為了自己內心的嚮往而不斷努力。

學習python

Q. 如果參與這個計畫，會怎麼安排學習時間？

由於我目前處於待業狀態，同時也在為轉職做準備，因此如果參加此次訓練，我會將大部分的時間都放在學習上，並且把它當作在模擬上班的情況來適應未來的工作。

如果每週需要投入 40 至 60 小時的學習時間，那麼我預計每天會花 7 至 8 小時在學習上。

為了保持學習的紀律，我會記錄一整天的時間花費，以了解時間都用在哪裡，讓我更有意識地掌握如何運用時間。

並且可以透過在第一階段的訓練期間先適應並記錄時間，不僅能掌握時間的運用，還能更加了解工作所需的時間，有助於後續的規劃。

Q. 是否有想要加入的軟體公司？為什麼想加入該公司？

在篩選目標公司時，我是往兩個方向去思考：

  1. 公司規模
  2. 產品走向

1. 公司規模

小型或新創公司通常員工較少，因此每個人需要負責的範圍較廣，需要學習更多的技能。同時，因為決策者較少，所以也會有更多參與決策的機會。

而中、大型公司有更多的員工，能夠學習的對象也更多。此外，中、大型公司在工作的分工通常較細，每個人負責的事情也相對單純，對於新人上手工作來說相對友善。

在面對第一份工作時，我會希望先踏穩腳步後穩定地成長與學習，所以中型公司對我可能是比較好的選擇。因為分工較細，工作上負責的事情相對單純，讓我比較能夠專注於自己負責的領域。

2. 產品走向

我也希望在一家有新產品、功能不斷創新改進的公司工作。這樣能夠不斷地學習和改進，也會有更多的機會成長。

Q. 請描述一件讓你產生明顯負面情緒的事情，你如何處理該情緒？

容易讓我產生負面情緒的事情

我容易在與他人討論一件事情（事實）時，若遇到明顯不符合邏輯（事實）的情況下產生焦躁感，甚至有可能陷入爭辯對錯的狀態。

當下如何處理該情緒

面對這種情況，我會盡量在被情緒控制前嘗試放下自身的立場和情緒，專注於討論事情本身。

後續處理

如果在當下有被情緒影響，我會在情緒緩和後，重新審視事情的經過，嘗試站在不同的角色和立場來思考問題的可能性和原因。

有時候也會刻意挑戰自己的觀點，因為我們的大腦很容易扭曲事實或提出有利於自己的觀點。

最後我會嘗試通過理性討論來解決問題，而不是僅僅為了證明自己的觀點而爭辯。

結語

我容易陷入在討論中的對錯辯論中，這個狀況已經伴隨我一段時間了。但是每次情緒緩和後，我都會反思這個過程。隨著時間的推移，我也越來越能夠在快要陷入情緒之前意識到這個問題。

雖然我還無法完全解決這個問題，但我相信我會越來越擅長處理這個問題。

Q. 關於這份申請網頁，請分享一個你開發時的技術心得。

透過這次實際製作簡單的靜態網頁，可以初步理解 HTML 標籤的意義和 CSS 元素的應用，並實作一些簡單的 JavaScript 功能。

在這個過程中遇到的一些重點與應用，我把它列舉在下方。

HTML 架構：

練習設計網頁的基本架構，各種標籤的使用方法和代表意義，並在實際製作中運用。

例如：<header>、<nav>、<section>、<main>、<footer>等標籤。

CSS 樣式設計：

學習如何調整元素的屬性，以實現網頁外觀的設計和特效。

例如：元素的淡入淡出效果、偽元素的畫面裝飾等。

JavaScript 運用：

運用基本語法處理簡單的 DOM 操作，完成所需的功能和效果。

例如：自製按鈕來達成想要的滑動效果、隱藏與顯現等。

RWD 設計：

針對不同的介面大小，來設計適合的網頁外觀和使用體驗。

例如：在桌機上使用 hover 效果來設計按鈕，在手機上則使用 click 效果，並進行不同的網頁排版設計等。
